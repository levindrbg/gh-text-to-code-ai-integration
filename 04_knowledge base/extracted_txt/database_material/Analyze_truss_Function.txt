Function: analyze_truss(...)Performs a linear‐elastic finite‐element analysis on the current truss.Global stiffness assemblyLoop each element, compute length L, direction cosines (c,s)Build local 4×4 stiffness k_e = (A·E/L) · [ … ]Scatter‐add k_e into the global K matrix at the appropriate DOFs.Load vectorAssemble the global force vector F from load_nodes / load_values.Apply BCs & SolvePartition out the fixed DOFs, solve K_ff · U_f = F_f for the free displacements U.Post‐processingCompute axial strains in each element from the nodal displacement differences.Compute axial stresses via Hooke’s law: σ = E·ε.def analyze_truss(nodes, elements, areas, materials, fixed_dofs, load_nodes, load_values): """Performs FEA to find displacements, strains, and stresses.""" num_nodes = len(nodes) num_elements = len(elements) num_dofs = num_nodes * 2 K = np.zeros((num_dofs, num_dofs)) element_lengths = np.zeros(num_elements) for i, (n1_idx, n2_idx) in enumerate(elements): n1, n2 = nodes[n1_idx], nodes[n2_idx] L = np.linalg.norm(n2 - n1) if L == 0: continue element_lengths[i] = L c = (n2[0] - n1[0]) / L s = (n2[1] - n1[1]) / L E = materials[i]['E'] A = areas[i] k_e = (A * E / L) * np.array([ [cc, cs, -cc, -cs], [cs, ss, -cs, -ss], [-cc, -cs, cc, cs], [-cs, -ss, cs, ss] ]) dofs = [2n1_idx, 2n1_idx+1, 2n2_idx, 2n2_idx+1] for row, dof_r in enumerate(dofs): if dof_r >= num_dofs: continue for col, dof_c in enumerate(dofs): if dof_c >= num_dofs: continue K[dof_r, dof_c] += k_e[row, col] F = np.zeros(num_dofs) for i, node_idx in enumerate(load_nodes): dof_x, dof_y = 2node_idx, 2node_idx + 1 if dof_y < num_dofs: F[dof_x] += load_values[i, 0] F[dof_y] += load_values[i, 1] free_dofs = np.setdiff1d(range(num_dofs), fixed_dofs) U = np.zeros(num_dofs) try: if K[np.ix_(free_dofs, free_dofs)].shape[0] > 0: U_reduced = np.linalg.solve(K[np.ix_(free_dofs, free_dofs)], F[free_dofs]) U[free_dofs] = U_reduced except np.linalg.LinAlgError: print("Warning: Singular matrix. Structure may be unstable.") return None, None, None, None strains = np.zeros(num_elements) for i, (n1_idx, n2_idx) in enumerate(elements): n1, n2 = nodes[n1_idx], nodes[n2_idx] L = element_lengths[i] if L == 0: continue c = (n2[0] - n1[0]) / L s = (n2[1] - n1[1]) / L u1x, u1y = U[2n1_idx], U[2n1_idx+1] u2x, u2y = U[2n2_idx], U[2*n2_idx+1] strains[i] = (1/L) * (c * (u2x - u1x) + s * (u2y - u1y)) stresses = np.array([materials[i]['E'] * strains[i] for i in range(num_elements)]) return U.reshape((num_nodes, 2)), strains, stresses, element_lengths
