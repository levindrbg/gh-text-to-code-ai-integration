Features:      -  Converts  mm  →  meters      -  Pads  2D→3D  coordinates      -  Validates  inputs      -  Generates  analysis-ready  CSVs       Returns:  Dict  of  output  file  paths      """  #  Input  validationif  not  nodes  or  not  elements:          raise  ValueError("Nodes  and  elements  lists  cannot  be  empty")       nodes  =  np.array(nodes)      if  nodes.ndim  !=  2  or  nodes.shape[1]  not  in  (2,3):          raise  ValueError("Nodes  must  be  Nx2  or  Nx3  array")       elements  =  np.array(elements)      if  elements.ndim  !=  2  or  elements.shape[1]  !=  2:          raise  ValueError("Elements  must  be  list  of  index  pairs")       if  np.any(elements  >=  len(nodes))  or  np.any(elements  <  0):          raise  ValueError("Element  indices  out  of  bounds")   #  ---  Member  Export  ---with  open(member_filename,  'w',  newline='')  as  f:
